# Sh.Framework
A collection of game related libraries that run alongside monogame

expect weekly stable releasese every Thursday.

## Currently featuring:
-Mouse and Keyboard stroke library (detect if a key on a keyboard has been pressed or released)
-A primitive entity component system and a screen system (to put all your objects in)
-Basic 2D physics
-Numerous sprite based UI and form templates

## Soon adding:
-Data serialization
-Basic audio and timing library
-Comprehensive debug tools
-User friendly visual novel tools (yes)

I'm very open to feedback and suggestions that aren't "make faster" or "codebase gay"
